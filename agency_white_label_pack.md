## Security & IP Protection

We are built to be a trusted partner. Our platform security is designed for agency and enterprise use.

- **Server-Side Keys:** All API keys (OpenAI, Resend, etc.) and database credentials are kept on the server and are never exposed to the client.

- **Role-Based Access Control:** The admin dashboard is protected by `adminProcedure` middleware, ensuring only authorized users can access lead data and platform stats.

- **Secure Data Handling:** We use a secure database with parameterised queries to prevent SQL injection. All data is handled in accordance with best practices.

- **Proprietary Engine:** Your agency and clients see the report output, but our core Python scoring engine and methodology remain proprietary to SMAIDM.

- **NDA & Non-Compete:** All white-label partnerships are governed by a mutual Non-Disclosure Agreement with clear non-compete and IP clauses to protect both parties.
